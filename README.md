# FlaskAPI-chat-application

## Installation
```
git clone https://github.com/Abbilaash/FlaskAPI-chat-application.git
```
```
cd FlaskAPI-chat-application
```
```
pip install -r requirements.txt
```

## Run
```
python app.py
```
